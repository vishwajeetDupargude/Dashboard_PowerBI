#  Cricket Best 11 - Power BI Dashboard

This project is a Power BI dashboard analyzing player performance in the T20 World Cup. 
It highlights key batting metrics such as Strike Rate, Batting Average, and Boundary %, 
and organizes players into roles like Power Hitters, Anchors, Finishers, All Rounders, 
and Specialist Fast Bowlers. The dashboard is interactive, with slicers for filtering 
by team, stage, and batting style, and visuals that make comparison easy.
